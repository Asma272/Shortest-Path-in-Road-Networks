# Shortest-Path-in-Road-Networks
Shortest Path in Road Networks

A graph analysis project that compares Dijkstra and Bellman-Ford algorithms using the California road network dataset (roadNet-CA).

Technologies
Python
NetworkX
Matplotlib
Jupyter Notebook

Project Workflow
Graph Construction
Dataset Preprocessing
Algorithm Implementation
Performance Analysis
Running Time Evaluation
Complexity Analysis

Algorithms
Dijkstra Algorithm
Bellman-Ford Algorithm

Evaluation
Execution Time Comparison
Best, Average, and Worst Case Analysis
Time Complexity Analysis
Space Complexity Analysis

Results

The experimental results showed that Dijkstra's algorithm consistently outperformed Bellman-Ford on large-scale road network datasets with non-negative edge weights.

This project was developed as part of a team-based university project.
